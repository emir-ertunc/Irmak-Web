## Faz Durumu

- [x] Site adı, kısa açıklama ve ana sayfa tonu belirlendi.
- [x] Pembe ana renk ailesi ve destek renkleri seçildi.
- [x] Çiçekli, cıvıl cıvıl, tatlı ve neşeli feminen atmosfer için moodboard kararları yazıldı.
- [x] Çiçek, kalp, parıltı, kurdele, sticker ve el yapımı hissi veren dekoratif motif listesi belirlendi.
- [x] Yazı tipi, başlık ölçekleri ve metin ağırlıkları belirlendi.
- [x] Buton, form, kart, rozet ve bildirim stilleri tasarlandı.
- [x] Canlı ama tek renge sıkışmayan renk dengesi için pembe dışı destek renklerin kullanım yerleri belirlendi.
- [x] Ürün fotoğrafı kullanım kuralları belirlendi.

## Renk Sistemi

- [x] Şeker pembe: `#F78FB3`

Kullanım kararları:

- [x] Pembe ana duygu rengi olacak, ancak tüm arayüz sadece pembe tonlarına sıkışmayacak.
- [x] Krem beyaz geniş arka planlarda nefes alan alan yaratacak.
- [x] Koyu vişne metin rengi başlık, açıklama ve fiyat okunurluğunu taşıyacak.
- [x] Nane yeşili başarı ve olumlu durumlarda kullanılacak.
- [x] Mercan dikkat isteyen çağrı butonlarında ve kısa vurgularda kullanılacak.
- [x] Limon sarısı parıltı, rozet ve yeni ürün vurgularında kontrollü kullanılacak.
- [x] Lavanta özel sipariş, koleksiyon ve yumuşak arka plan detaylarında kullanılacak.

Kontrast kuralları:

- [x] Açık pembe üstünde açık metin kullanılmayacak.
- [x] Buton metinleri koyu vişne veya beyaz ile yeterli kontrast sağlayacak.
- [x] Fiyat, stok ve sipariş durumu gibi kritik bilgiler dekoratif renklerle okunmaz hale getirilmeyecek.

- [x] Admin panelinde dekor dozu azaltılacak, marka rengi sadece vurgu olarak kullanılacak.

## Dekoratif Motif Listesi

Ana motifler:

- [x] Minik çiçek
- [x] Kalp
- [x] Parıltı
- [x] Kurdele
- [x] Sticker etiketi
- [x] Yuvarlak boncuk
- [x] Minik yıldız
- [x] El yazısı hissi veren kısa not etiketi

Kullanım yerleri:

- [x] Çiçek: ana sayfa arka plan detayları, koleksiyon başlıkları, ürün kartı hover hareketi.
- [x] Kalp: favori, teşekkür ekranı, özel sipariş notları.
- [x] Parıltı: yeni ürün, sepete ekleme, başarı bildirimi.
- [x] Kurdele: paketleme, hediye seçeneği, özel koleksiyon rozetleri.
- [x] Sticker etiketi: rozet, stok az, yeni ürün, özel sipariş.
- [x] Boncuk ve yıldız: sayfa geçişleri, küçük boş durum görselleri.

Kısıtlar:

- [x] Aynı ekranda üçten fazla dekoratif motif tipi baskın kullanılmayacak.
- [x] Motifler ürün görsellerinin üzerine kalıcı olarak bindirilmeyecek.
- [x] Sürekli hareket eden motif sayısı düşük tutulacak.

## Tipografi

Başlık yazı karakteri yönü:

- [x] Yumuşak, karakterli ve tatlı bir başlık hissi.
- [x] Önerilen aile: `Fraunces` veya benzer yuvarlak serif karakterli başlık yazısı.

Gövde yazı karakteri yönü:

- [x] Okunabilir, sıcak ve mobilde rahat taranan bir yazı hissi.
- [x] Önerilen aile: `Nunito` veya benzer yuvarlak sans-serif karakterli gövde yazısı.

Yedek sistem:

- [x] Başlık: `Georgia`, `serif`
- [x] Gövde: `Arial`, `sans-serif`

Başlık ölçekleri:

- [x] Hero başlık: 40-56px masaüstü, 32-40px mobil.
- [x] Sayfa başlığı: 32-40px masaüstü, 28-32px mobil.
- [x] Bölüm başlığı: 24-32px masaüstü, 22-28px mobil.
- [x] Kart başlığı: 16-18px.
- [x] Admin paneli başlıkları: 20-28px, daha sade ve sıkı.

Metin ağırlıkları:

- [x] Hero ve sayfa başlığı: 700.
- [x] Bölüm başlığı: 650-700.
- [x] Gövde metni: 400-500.
- [x] Buton ve rozet: 650.
- [x] Fiyat: 700.

Fotoğraf tonu:

- [x] Aydınlık, sıcak ve yumuşak.
